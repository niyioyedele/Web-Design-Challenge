Summary: Latitude vs. X


In this project we are analyzing how weather changes as you get closer to the equator. Our dataset of over 500 cities is pulled from the OpenWeather API we first pulled data from the OpenWeatherMap API to assemble a dataset of over 500 cities.

After assembling the dataset, we used MatplotLib to plot various aspects of the weather vs latitude. We looked at, temperature, cloudiness, humidity, and wind speed. This site provides the source data and charts created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
